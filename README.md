# Cloud-Based Secure Task Management Web Application

This project is a secure task management web application developed using Flask and deployed as part of a DevSecOps capstone project.

## Features
- User registration and login
- Password hashing
- Role-based access control
- User dashboard
- Admin dashboard
- Task creation and deletion
- SQLite database

## Technologies Used
- Python Flask
- HTML/CSS
- SQLite
- Git and GitHub
- AWS EC2
- Nginx
- CloudWatch

## Security Features
- Password hashing using Flask-Bcrypt
- Session-based authentication
- Role-based access control
- SQL injection prevention using parameterized queries
- XSS protection using Jinja2 template escaping

## Project Objective
To design, build, secure, deploy, and monitor a cloud-based web application using DevSecOps best practices.
